# PRODIGY_SD_01

## Task-01: Temperature Conversion Program

This project is completed as part of the **Prodigy Software Development Internship**.

 ###Task Description
The objective of this task is to create a program that converts temperatures between:
- Celsius
- Fahrenheit
- Kelvin
The program accepts a temperature value and its unit from the user and converts it into the other two units
 ###Technologies Used
- Python
- Visual Studio Code
 ###How to Run the Program
1. Install Python 3
2. Open the INTERNSHIP_TASKS folder in VS Code
3. Run the program using the command:
4.sample input:Enter the temperature value: 25
Enter the unit (Celsius, Fahrenheit, Kelvin): celsius
5.sample output:
       Fahrenheit: 77.00
       Kelvin: 298.15
