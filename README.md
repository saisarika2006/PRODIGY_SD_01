# PRODIGY_SD_01
This project was completed as part of the **Prodigy Software Development Internship**.
## Task Description
The objective of this task is to create a program that converts temperatures between:
- Celsius
- Fahrenheit
- Kelvin
The program accepts a temperature value and its unit from the user and converts it into the other two units.
## Technologies Used
- Python
- Visual Studio Code
## How to Run the Program
1. Install Python on your system.
2. Open the project folder in VS Code.
3. Run the program using:
### Sample Input
Enter the temperature value: 25
Enter the unit (Celsius, Fahrenheit, Kelvin): Celsius
### Sample Output
Fahrenheit: 77.00
Kelvin: 298.15
