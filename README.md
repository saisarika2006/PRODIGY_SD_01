# PRODIGY_SD_01 – Temperature Conversion Program

## 📌 Overview

This project was developed as part of the **Prodigy InfoTech Software Development Internship**.

The Temperature Conversion Program is a Python-based application that converts temperature values between **Celsius**, **Fahrenheit**, and **Kelvin**. The program accepts a temperature value and its unit as input and automatically calculates the equivalent values in the other temperature scales.
## 🎯 Objective

The objective of this task is to understand and implement:

* User input handling
* Conditional statements
* Mathematical calculations
* Temperature conversion formulas
* Basic Python programming concepts
## ✨ Features

* Convert Celsius to Fahrenheit and Kelvin
* Convert Fahrenheit to Celsius and Kelvin
* Convert Kelvin to Celsius and Fahrenheit
* User-friendly console interaction

## 🔢 Temperature Conversion Formulas
### Celsius to Fahrenheit
```text
°F = (°C × 9/5) + 32
```
### Celsius to Kelvin
K = °C + 273.15
```
### Fahrenheit to Celsius
°C = (°F − 32) × 5/9
```
### Kelvin to Celsius
°C = K − 273.15
